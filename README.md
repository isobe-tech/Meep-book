# Meep-book (code)

This repository hosts the Python code used in the book “Introduction to Electromagnetic Simulation with Meep”.

## Contents

- `code/`: Chapter scripts (Chapter 6–9)
- `scripts/`: Figure/regeneration helper scripts
- `cad/`: CAD assets used by some Chapter 8 examples
- `environment.yml`: Conda environment for PyMeep examples
- `environment_cad.yml`: Conda environment for CAD-related tooling (optional)

## Setup (Conda)

- Create environment: `conda env create -f environment.yml`
- Activate: `conda activate meep-book`

## Run examples

- `python code/ch06/hello_meep_wave2d.py`
